DROP Database Statement:

    * DROP DATABASE statement is used to delete a database along with all the data such as tables, views, indexes, stored procedures, and constraints.

Syntax: 

    +------------------------------+
    | DROP DATABASE DatabaseName;  |
    +------------------------------+

Dropping a Database using mysqladmin:

    +----------------------------------------------------+
    | [root@host]# mysqladmin -u root -p drop TUTORIALS  |
    | Enter password:******                              |
    +----------------------------------------------------+
